Hospital Management System API

Welcome to the Hospital Management System API documentation. This API provides a comprehensive set of endpoints for managing hospital operations, including:

  • Patient management
  • Doctor management
  • Medical history tracking
  • Vital signs monitoring
  • Insurance information
  • And more...

The API uses JWT authentication for secure access. Different roles (Admin, Doctor, Patient, User) have different access levels to various endpoints.

Go to Swagger UI Documentation

For API requests, you can use the base URL: http://localhost:8081
